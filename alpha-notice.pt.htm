Avisos

  • Stepmania 5.3 ainda está em desenvolvimento, então provavelmente vai ter bugs e falhas. Se você encontrar qualquer problema, quer fazer um comentário ou tem alguma questão, por favor reporte ou discuta sobre eles no nosso Github para rastreamento público de problemas, servidor do Discord ou fórum. NÃO REPORTE PROBLEMAS PARA ESSAS VERSÕES NO GITHUB PRINCIPAL DO STEPMANIA..
  • Se estiver atualizando de uma versão antiga, desinstale-o e delete a pasta Appearance/Theme na pasta do jogo antes de instalar a próxima versão, ou instale em outra localização. Atualizações em versões já instaladas podem criar problemas enquanto o jogo é executadp.
  • Mudanças no sistema de entrada no StepMania 5.3 podem ter impactado sua habilidade de usar controles sem configurações adicionais. Veja o guia de controle para mais informações
  • Para versões ARM, digite uname -a no shell para determinar a arquitetura correta para você baixar.
