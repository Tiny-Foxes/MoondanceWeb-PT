Avisos

  • Stepmania 5.3 ainda está em desenvolvimento, então provavelmente haverão bugs e falhas. Se você encontrar quaisquer problemas, quiser fazer um comentário ou tirar dúvidas, por favor reporte ou discuta sobre eles em nossa página do Github para solução de problemas, servidor do Discord ou fórum. NÃO REPORTE PROBLEMAS PARA ESSAS VERSÕES NO GITHUB PRINCIPAL DO STEPMANIA.
  • Se estiver atualizando de uma versão antiga, desinstale-a e delete a pasta Appearance/Theme na pasta do jogo antes de instalar a próxima versão, ou instale em outra pasta. Atualizações por cima de versões já instaladas podem causar problemas enquanto o jogo é executado.
  • Mudanças no sistema de entrada no StepMania 5.3 podem ter impactado sua habilidade de usar controles sem configurações adicionais. Veja o guia de controles para mais informações
  • Para versões ARM, digite uname -a no shell para determinar a arquitetura correta para você baixar.
